## API

### Code Props

| Attribute | Type                                                                        | Description              | Default   |
| --------- | --------------------------------------------------------------------------- | ------------------------ | --------- |
| children  | `ReactNode`                                                                 | The content of the code. | -         |
| size      | `xs` \| `sm` \| `md` \| `lg` \| `xl`                                        | The size of the code.    | `md`      |
| color     | `default` \| `primary` \| `secondary` \| `success` \| `warning` \| `danger` | The color of the code.   | `primary` |
| radius    | `none` \| `base` \| `xs` \| `sm` \| `md` \| `lg` \| `xl` \| `full`          | The radius of the code.  | `md`      |
