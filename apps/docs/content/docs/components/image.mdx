### APIs

<Spacer y={2} />

#### Image Props


| Attribute            | Type                                            | Accepted values                 | Description                                          | Default      |
| -------------------- | ----------------------------------------------- | ------------------------------- | ---------------------------------------------------- | ------------ |
| **src**              | `string`                                        | -                               | Image source (local or remote)                       | `-`          |
| **autoResize**       | `boolean`                                       | `true/false`                    | Resize Image to fits screen                          | `false`      |
| **showSkeleton**     | `boolean`                                       | `true/false`                    | Shows loading Skeleton while image is loading        | `true`       |
| **objectFit**        | `ObjectFit`                                     | [ObjectFit](#image-objectfit)   | Property tells the content to fill the container     | `scale-down` |
| **width**            | `string`                                        | `string`                        | Specifies Image Width                                | `-`          |
| **height**           | `string`                                        | `string`                        | Specifies Image Height                               | `-`          |
| **maxDelay**         | `number`                                        | -                               | Specifies how long ShowSkeleton Renders Animation    | `3000`       |
| **className**        | `string`                                        | `string`                        | Give Image A ClasName                                | `-`          |
| **containerCSS**     | `Stitches.CSS`                                  | -                               | Override Default Image Contianer Style               | `-`          |
| **css**              | `Stitches.CSS`                                  | -                               | Override Default CSS style                           | `-`          |
| **as**               | `keyof JSX.IntrinsicElements`                   | -                               | Changes which tag component outputs                  | `a`          |
| ...                  | `AnchorHTMLAttributes`                          | `'rel', 'target', ...`          | Native props                                         | `-`          |
