## Button

<Text>
  Buttons allow users to perform actions and choose with a single tap.
</Text>

```jsx
import { Button } from '@nextui/react';
```

<Playground
  title="Default"
  desc="The default `Button` contains an animation effect."
  code={`
<Button>Default</Button>
`}
/>

<Playground
  title="Disabled"
  desc="Unusable and un-clickable `Button`."
  code={`
<Button disabled>Default</Button>
`}
/>

<Playground
  title="Shadow"
  desc="You can add a shadow effect with the property `shadow`."
  code={`
 <Button shadow color="secondary">Shadow</Button>
`}
/>

<Playground
