#### Spacer Props

| Attribute  | Type               | Accepted values          | Description    | Default |
| ---------- | ------------------ | ------------------------ | -------------- | ------- |
| **x**      | `number` / `float` | -                        | x-axis spacing | 1       |
| **y**      | `number` / `float` | -                        | y-axis spacing | 1       |
| **inline** | `boolean`          | -                        | inline space   | `false` |
| ...        | `HTMLAttributes`   | `'id', 'className', ...` | native props   | -       |
